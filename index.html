Mission

Founded in a time when software becomes more short-sighted and controlled. the web becomes more closed. software companies recklessly hire and fire employees.

We envision a world where software the web software company is more ethical.

Ethical-ware

Ethical-ware is software that promotes better longevity, more user agency, and freer flow of information.

Transparency

We believe transparency is key to ethical-ware, hence we are going to:

  • Use and author open-source software.
  • Open our financial.

Hosting

We provide the following ethical-ware based web services for free:

  • Mastodon is a decentralized social network.
  • RSSHub is an open-source webapp that converts web sites into RSS feeds.
  • Nitter an alternative Twitter front-end.
  • rss-lambda is a webapp that performs lambda on your RSS feeds.

Donations are optional but appreciated.

Funds are used to cover hosting and labor costs.

Consulting

We provide consulting services around ethical-ware:

  • Software development of new and existing ethical-ware.
  • Software infrastructure deployment and support for ethical-ware based web services.

Software development philosophy

  • Design for better longevity.
  • Build on existing standards, protocols and platforms.
  • Keep it simple and stupid.

A sustainable software company

A more ethical software company is first more sustainable, hence we are going to:

  • Start new projects or expand scope cautiously.
  • Control infrastructure costs at all time, instead of an afterthought.
  • Strive for getting bootstrapped instead of relying on external investment.
  • Hire cautiously.
